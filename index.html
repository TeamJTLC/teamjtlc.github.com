Java Thin Layer Chromatography
A wizard analyzer for thin layer chromatography.

About

jTLC is a software devised to help and assist the chromatographic analysis of samples of some oils like biodiesel, from images of base plates on which analysis experiments were made.

The software processes the image and calculates the mean for each sample in the sources. Using the mean data, it computes the position of each component of the compound and the region that it occupies, which is then integrated to compute the superficial area of the corresponding region.

Using the area values and the total surface, the software computes the relative area of each component and generates reports with the results of this process.

jTLC

This is the main interface of jTLC client. Here you can start a new project, load previous projects, change system settings and more!

Project Creation

The new project dialog allows the user to keep track of the experiment dates, title and description for future reference.

Source Image

To start a new project, just load an experiment image by a simple drag & drop or through the file explorer dialog.

Image Crop

This allows the user to crop invalid areas of the experiment image. These areas are initially computed by the system.

Image Rotation

This step allows one to rotate source images to fix samples alignments. It also brings tools for images flips and inversions.

Samples Selection

In this step the user can select the individual samples using simple sliders over the source image. The initial positions of these slider thumbs are computed by the system process wizard.

Samples Special Points

The wizard splits the samples into individual images and allows the user to set specific data for each one, including the seeding point and the solvent front. Each point can be specified individually for each sample, or globally. This step also allows one to set the name and particular comments for each sample.

Samples Analysis

The analysis of samples is carried out in a per-sample basis. In this step the user can select each peak position for each of the samples. Initially, the positions of the peaks are computed by the process wizard.

Samples Comparation

In this tab the user can compare each sample mean in one image. This view allows the user to change each sample color and export the plot graph as an external image.

Samples Peaks Analysis

After the sample peaks selection, the system computes each peak surface and the sample total surface (all peaks surface) and shows a summary of each peak total surface, relative surface, position, etc.

Analysis Reports

After the samples analysis, the system generates reports where the results are grouped by samples and by experiment. Here, the user can visualize the results of the experiment analysis for each sample and for each peak. The system also allows the user to export the results/reports as PDF, ODT, HTML and others formats.

Proyects Gallery

In the gallery panel the user can explore all the projects created and stored in a folder in the file system and continue their analysis.
