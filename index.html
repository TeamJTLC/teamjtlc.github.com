jTLC

This is the main interface of jTLC client. Here you can start a new proyect, load previous proyects, change sistem settings and more!

Project Creation

The new project dialog allows to the user keep a track of the experiment dates, title and description for future use.

Source Image

Starting a new project, load experiment image by a simple drag & drop or through the file explorer dialog.

Image Crop

This allows to the user to crop invalid areas of the experiment image. These areas are intially computed by the system.

Image Rotation

This step allows to rotate source images to fix samples alignments. Also brings tools for images flips and inversions.

Samples Selection

In this step the user can select the individual samples using simples sliders over the source image. The initial positions of these slider thumbs are computed by the system process wizard.

Samples Special Points

The wizard split the samples in individual images and allows to the user to set the specific data of each one, like the seed point and the solvent front. Each point can be specific individually or globally for each samples. This step also allows to set the samples name and specific comments for each one.

Samples Analysis

The samples analysis is carried out individually for each one. In this step the user can select each peak position for each samples. Initially the peaks positions is computed by the process wizard.

Samples Comparation

In this tab the user can compare each sample mean in one image. This view allows to the user to change each sample color and export the plot graph as external image.

Samples Peaks Analysis

After the samples peaks selections, the system compute each peak surface and the sample total surface (all peaks surface) and show a summary of each peak total surface, relative surface, positions and others.

Analysis Reports

After the samples analysis the system generate reports where the results are groupped for each samples and for the experiment. Here the user can visualize the results of the experiment analysis for each samples and for each peak. The system also allows to the user to export the results/reports as PDF, ODT, HTML and others formats.

Proyects Gallery

In the gallery panel the user can explore all the projects created and stored in a folder in the file system and continue analysing of them.
